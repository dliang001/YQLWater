V1.0    --   2017-04-25
1、统计界面 -- 查询标签： 查询后的信息增加"户名" 和 "户号" 两行
2、统计界面 -- 查询标签：  （1）缴费金额 --> 交费金额  （2）本次余额 --> 本次结余
3、上传界面 -- 上传电话、标签号、缴费，成功后，会同时将本地数据库的修改。
4、抄表界面 -- 扫描抄表后,上月结余 -- 重新根据本"标签号"获取对应的"上月结余"
5、抄表界面 -- 本月预交改为：本月总预交 + 本次预交
6、抄表界面 -- 点击"保存"后： 本月结余 = 上月结余 + 本月总预交 - 本月应收
7、抄表界面 -- 点击"预交水费"后： 本月结余 = 上月结余 + 本月总预交 - 本月应收
                本月总预交 = 之前的本月总预交 + 本次预交
                同时，将本次预交上传到服务器
8、抄表界面 -- 扫描抄表，在"预交水费"重新获取电子标签，放到全局变量。
9、抄表界面 -- 点击"预交水费"后，如果预交了，会提示是否继续预交
10、统计界面 -- 新增："未抄表"标签，根据月份查出"未抄表"的数据
